# Camelot Keys

A visual harmonic mixing wheel for DJs based on the Camelot system.

## Usage

Open `index.html` in any browser. No build step or dependencies required.

## Features

- **Interactive wheel** - Click any key to select it
- **Compatibility highlighting** - Shows compatible keys for smooth transitions:
  - **Same** (green) - Identical key
  - **Adjacent ±1** (blue) - Energy shifts up/down
  - **Relative A↔B** (pink) - Major/minor mood switch
  - **Parallel ±3** (orange) - Dramatic key change
- **Example journey** - Pre-built mixing path demonstration
- **All keys reference** - Quick lookup grid

## Mixing Tips

- **+1 clockwise** = build energy
- **-1 counter-clockwise** = cool down
- **A↔B switch** = mood shift (minor↔major)
